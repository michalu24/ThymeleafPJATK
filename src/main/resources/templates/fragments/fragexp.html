~{...} – Fragment Expressions

Wyrażenia ~{...} służą do wstawiania lub osadzania fragmentów innych szablonów HTML. Umożliwiają ponowne wykorzystanie widoków, dzielenie strony na nagłówki, stopki, panele boczne czy komponenty formularzy. Fragmenty mogą być ładowane za pomocą th:insert, th:replace lub th:include.

Typowe zastosowania:

  • th:insert="~{fragments/header :: nav}" – wstawienie nagłówka
  • th:replace="~{fragments/footer :: copy}" – pełne zastąpienie elementu
  • th:include="~{fragments/info :: alert(type='info')}" – wstawienie fragmentu z parametrami

🧪 Przykład użycia fragmentu:

  <!DOCTYPE html>
  <html xmlns:th="http://www.thymeleaf.org">
  <head>
    <meta charset="UTF-8">
    <title>Strona z fragmentem</title>
  </head>
  <body>

    <div th:replace="~{fragments/header :: nav}"></div>

    <main>
      <h1>Treść strony głównej</h1>
    </main>

    <div th:replace="~{fragments/footer :: copy}"></div>

  </body>
  </html>

📁 Fragment: fragments/header.html

  <!DOCTYPE html>
  <html xmlns:th="http://www.thymeleaf.org">
  <body>

    <div th:fragment="nav">
        <nav class="bg-blue-500 text-white p-4">
            <a href="/" class="mr-4 hover:underline">Start</a>
            <a href="/about" class="hover:underline">O nas</a>
        </nav>
    </div>

  </body>
  </html>

📁 Fragment: fragments/footer.html

  <div th:fragment="copy">
      <footer class="bg-blue-600 text-white text-center py-4">
          © 2025 Thymeleaf Aplikacja. Wszelkie prawa zastrzeżone.
      </footer>
  </div>

🧩 Fragment kontrolera Spring Boot:

  @GetMapping("/")
  public String showHomePage() {
      return "index"; // szablon główny, który ładuje fragmenty
  }
